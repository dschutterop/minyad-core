# Security Policy

Please report security issues privately before opening a public issue.

Responsible disclosure contact: Daniel via The Ice Cream Engineer project channel, or the private contact address published with the repository owner profile.

Do not include secrets, real network topology, tokens, certificates, private keys, OAuth credentials, or device owner tokens in issues, pull requests, logs, screenshots, or example configs. If a secret was committed anywhere in history, remove it and rotate it at the source.
